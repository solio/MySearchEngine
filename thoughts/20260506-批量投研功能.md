## 思考过程

### 股票列表文件格式

每行格式：`股票代码|股票名称`

示例：
```
601012|隆基绿能
002594|比亚迪
300750|宁德时代
```

### 批量投研流程

1. 读取股票列表文件
2. 逐行解析股票代码和名称
3. 对每个股票执行搜索（普通+定向）
4. 保存搜索结果到search_results目录
5. 记录运行日志

### 目录结构调整

原结构：
```

├── config/

├── models/
├── search_results/
├── thoughts/

```

### 功能说明

batch.py 支持的参数：
- `--file <path>`: 股票列表文件路径（必填）
- `--targeted`: 是否仅定向搜索（可选，默认否）
- `--debug`: 是否输出详细信息（可选，默认否）
- `--mock`: 是否使用mock模式（可选，默认否）

### 输出结构

搜索结果按原逻辑保存，每个股票生成单独的搜索结果文件。
