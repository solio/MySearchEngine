# 20260506 - 支持精确到日查询

## 本次更新内容

1. 更新 `skill/skill.py` 中的 `query_history()` 函数，支持查询参数：
   - 只提供年：返回某年所有数据
   - 提供年+月：返回某年某月的数据
   - 提供年+月+日：返回精确某天的数据
2. 更新 `skill/skill.md` 文档，说明新的参数选项
3. 更新相关代码保持功能一致
4. git提交并推送代码

### 命令行接口设计

```bash
# 查询某年
python -m skill history 2026

# 查询某年某月
python -m skill history 2026 4

# 查询某年某月某日
python -m skill history 2026 4 30
```

### 目录遍历逻辑

根据参数判断：
1. 只有年，遍历该年所有月日目录
2. 有年和月，遍历该年该月的所有日目录
3. 有年月日，只遍历该日目录
